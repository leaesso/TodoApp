# TodoApp – C#

Application console simple pour gérer des tâches.

## Fonctionnalités
- Ajouter une tâche
- Lister les tâches
- Marquer une tâche comme terminée
- Supprimer une tâche
- Sauvegarde locale en JSON

## Technologies
- C#
- .NET 8
- Application Console

## Objectif
Projet portfolio : publier un projet par semaine pour progresser en C# et .NET.
